CSS - Flexbox

Flexbox provides a way to align and distribute space among items in a container, even when their sizes are unknown or dynamic.

 
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Output:

Box 1
Box 2
Box 3

CSS - Grid Layout

CSS Grid Layout enables a two-dimensional layout system for web pages, allowing you to create complex layouts easily.

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 15px;
}

Output:

Item 1
Item 2
Item 3
Item 4

CSS - Animations

CSS animations allow elements to gradually change from one style to another, providing smooth transitions and effects.

@keyframes slide {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}

.box {
  animation: slide 2s infinite;
}

Output:

Animated Box
