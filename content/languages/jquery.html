jQuery - Simplified JavaScript

jQuery makes JavaScript coding more concise and browser-compatible, offering easy-to-use functions and effects.

$("p.intro").css("font-weight", "bold");

Output:

This introductory paragraph is now bold using jQuery.

Selectors

jQuery selectors allow you to easily select and manipulate HTML elements.

$(".highlight").css("background-color", "#f9c74f");

Output:

This paragraph will be highlighted.

Handling Events

jQuery simplifies event handling like click, hover, and keypress.

$("#clickMe").click(function() { $(this).text("Clicked!"); });

Special Effects

jQuery provides effects like hide, show, fade, and slide to create dynamic pages.

$("#box").fadeOut(1000);

Manipulating DOM

jQuery can easily add, remove, or change HTML elements and content.

$("#addText").append("<p>New paragraph added!</p>");

Output:

Original content.

Plugins & Slider Example

jQuery supports many plugins, including sliders for interactive content.

$("#slider").slideUp().slideDown();

Output:

Slider Content
