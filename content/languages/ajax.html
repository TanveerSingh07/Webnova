AJAX - Asynchronous Data Loading

AJAX enables fetching data in the background without reloading the page.

$.ajax({
  url: "https://jsonplaceholder.typicode.com/posts/1",
  success: function(result) {
    alert(result.title);  // Alert user with the title of the post
  }
});

Concept of AJAX

AJAX stands for Asynchronous JavaScript and XML. It allows web pages to update data without reloading the whole page.

  // Basic AJAX call with callback
  $.ajax({
    url: "https://jsonplaceholder.typicode.com/posts/1",
    success: function(result) {
      console.log("Title:", result.title);
    }
  });

Click the button to log the title of the post in the console.

Using XMLHttpRequest

AJAX can also be used with the native XMLHttpRequest object for making requests.

  var xhr = new XMLHttpRequest();
  xhr.open('GET', 'https://jsonplaceholder.typicode.com/posts/1', true);
  xhr.onload = function() {
    if (xhr.status === 200) {
      var response = JSON.parse(xhr.responseText);
      console.log("Title:", response.title);
    }
  };
  xhr.send();

Sending Data Using GET Method

With AJAX, data can be sent via the GET method by appending parameters to the URL.

  $.ajax({
    url: "https://jsonplaceholder.typicode.com/posts",
    type: "GET",
    data: { userId: 1 },
    success: function(response) {
      console.log(response);
    }
  });

Sending Data Using POST Method

With AJAX, data can also be sent using the POST method to create or update resources.

  $.ajax({
    url: "https://jsonplaceholder.typicode.com/posts",
    type: "POST",
    data: {
      title: "New Post",
      body: "This is a new post created using AJAX.",
      userId: 1
    },
    success: function(response) {
      console.log("New Post Created:", response);
    }
  });
