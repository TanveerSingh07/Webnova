HTML - Structure of the Web

HTML provides the skeleton of a webpage using tags and elements.

<h1>Hello World</h1>

Output:

Hello World

Adding an Image

You can display images using the <img> tag.

<img src="image.jpg" alt="A beautiful scenery">

Output:

Creating Lists

Use <ul> for unordered lists and <ol> for ordered lists.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ul>
  
  <ol>
    <li>First</li>
    <li>Second</li>
    <li>Third</li>
  </ol>

Output:

  • Item 1
  • Item 2
  • Item 3
  1. First
  2. Second
  3. Third

Creating a Table

Tables are made using <table>, <tr>, <th>, and <td>.

<table border="1">
    <tr>
      <th>Name</th>
      <th>Age</th>
    </tr>
    <tr>
      <td>Alice</td>
      <td>24</td>
    </tr>
    <tr>
      <td>Bob</td>
      <td>30</td>
    </tr>
  </table>

Output:

Name Age
Alice 24
Bob 30

Creating a Basic Form

Forms collect user input with tags like <input> and <button>.

<form>
    <label>Name:</label>
    <input type="text"><br>
    <label>Email:</label>
    <input type="email"><br>
    <button type="submit">Submit</button>
  </form>
