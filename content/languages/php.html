PHP - Server-Side Power

PHP is a popular server-side scripting language designed for web development. It can dynamically generate content, handle forms, manage sessions, and interact with databases.

<?php
$greeting = "Hello, World!";
echo $greeting;
?>

Output:

Hello, World!

Introduction to PHP and Variables

PHP code is written between <?php ?> tags. Variables in PHP start with a $ sign.

<?php
$name = "Alice";
$age = 25;
echo "Name: $name, Age: $age";
?>

Output:

Name: Alice, Age: 25

Control Flow in PHP

PHP supports control structures like if-else and loops.

<?php
$score = 85;
if ($score >= 50) {
  echo "Pass";
} else {
  echo "Fail";
}
?>

Output:

Pass

Working with Arrays

Arrays store multiple values. Use loops to iterate over them.

<?php
$colors = ["red", "green", "blue"];
foreach ($colors as $color) {
  echo $color . " ";
}
?>

Output:

red green blue

Functions and Object-Oriented PHP

PHP supports both user-defined functions and OOP with classes and objects.

<?php
function greet($name) {
  return "Hello, $name!";
}
class Person {
  public $name;
  function __construct($name) {
    $this->name = $name;
  }
}
echo greet("Alice");
$person = new Person("Bob");
echo $person->name;
?>

Output:

Hello, Alice! Bob

Using PHP with MySQL

PHP can connect to MySQL to perform database operations.

<?php
$conn = mysqli_connect("localhost", "root", "", "testdb");
if ($conn) {
  echo "Connected to DB";
} else {
  echo "Connection failed";
}
?>

Output:

Connected to DB
